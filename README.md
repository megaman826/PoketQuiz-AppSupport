# 앱 사용 방법

1. 앱 실행
2. 메인 화면에서 드롭다운 메뉴를 통해 원하는 몬스터 세대를 선택
3. 화면 키보드를 사용해 몬스터 이름 입력
4. 가능한 한 빠른 시간 안에 모든 몬스터의 이름을 맞추는 것이 목표
5. 더 이상 정답이 떠오르지 않을 경우 “정답 확인” 버튼을 눌러 정답 확인 가능
6. 리셋 버튼을 통해 언제든 현재 화면을 초기화 가능
7. 종료 후 결과 화면에서 최고 기록 시간(Best Time) 확인
8. 설정 팝업에서 정답 입력 시 재생되는 효과음 변경 가능

로그인, 숨겨진 메뉴, 외부 이동 과정은 없습니다.

**Platform:** AOS/iOS

**Description:**

PokeQuiz is a fast-paced time-attack quiz game where players must correctly type the names of collectible monsters within a limited time. The game focuses on memory, recall speed, and knowledge across multiple generations of monster collections. The app is fully playable offline and does not require user accounts or personal data.

# How to Use

1. Launch the app.
2. On the main screen, select a monster generation using the dropdown menu.
3. Enter monster names using the on-screen keyboard.
4. The objective is to correctly input the names of all monsters as quickly as possible.
5. If the player cannot remember further answers, they can tap the “Show Answers” button to view the correct answers.
6. The Reset button can be used at any time to reset the current session.
7. After completing the session, the results screen displays the best completion time.
8. In the Settings popup, players can change the sound effect that plays when a correct answer is entered.

No login, hidden navigation, or external links are required.
